# 总览

（待完善）

本文档介绍了 EB tresos Studio 中部分 Module 的使用，参考 `<your_rtd_address>/eclipse/plugins/<module_name>/doc/RTD_xxx_UM.pdf` 编写。

> Described: Base, Mcu, Port, Dio

**RTD版本号为 1.0.0。**

在阅读之前，请务必按照 [环境配置](../环境配置.md) 和 [EB Tresos Mcal 最小系统搭建](../EB%20Tresos%20Mcal%20最小系统搭建.md) 完成相应工程配置。

需要注意的是，由于时间限制，本说明对于大多数模块并没有覆盖所有的设置项，而是基于工程实践和车队应用场景进行选择性诠释。更全面的介绍请参考 UM。

下面是对于各个模块 User Manual 结构的介绍（以 Base 为例）：

1. Introduction
    
    1. 支持的 Derivatives （可以理解为支持的芯片型号）
    2. 对于 AUTOSAR 的介绍
    3. 对于文档的相关规定
    4. 术语介绍，如缩写等
    5. 其他
2. Driver
    
    1. 代码组成
    2. 对各个头文件的描述
    3. 限制（Base 中留空）
    4. NXP等芯片厂商的标准实现与标准的不同（not fully implemented）
    5. 配置提示，包括对于某些代码依赖库的配置提示（如在 Base 中，包括了对 stdint.h 依赖的相关提示）
    6. 可能抛出的 DEM 错误
    7. 其他
3. Tresos Configuration Plug-in
    
    该部分详细阐述了在 EB tresos Studio 中配置 Mcal 所需要的相关参数，与 EB tresos 中的设置项高度对应。
    
4. Others
