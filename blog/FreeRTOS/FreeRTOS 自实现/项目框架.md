# 项目框架

工程分为三个目录：

- LinRTOS
- Portable
- User

其中 LinRTOS 为 RTOS Kernel 主要实现，Portable 为移植代码，User 中为 main.c 文件。

README 中给出的工程文件结构如下：

```c
├─Code
│  ├─LinRTOS Kernel     // RTOS内核文件
│  │  ├─inc             // 内核头文件
│  │  └─portable        // RTOS调用的硬件配置
│  │      └─RVDS
|  |  src               // RTOS源文件
│  ├─RTE
│  │  ├─Device          // 硬件支持包
│  └─User  
```
